# Propt: CI/CD Recommendations

## Input

What are modern CI/CD tooling to be used for a modern front-end monorepo library that contains:

- theme package with css and design tokens written in Typescript.
- core package with common utilties, authentication, permissions checks written in Tyepscript.
- web-components package written in Lit with React and Angular wrappers.
- Additional libraries and applications will be added.

Tools and libraries used include:

- Bun
- Vite
- OXC
- TurboRepo
- Changeet

There is a Dockerfile and Jenksfile in place.

Compare the options on Github including Github Actions, Github packages and other library or technology for CI/CD and can publish to npm registry.
Give the pros and cons for each and a final recommendation for this repo.

## Output

Create a markdown file summarizing the pros and cons of each modern approach and a final recommendation.
Create the markdown file in the path + name /prompts/CICD_RECOMMENDATIONS.md.
