# @oxi-ui/theme

## 0.0.1

### Initial Release

- Light and dark theme support using modern CSS `light-dark()` syntax
- CSS custom properties for all design tokens
- TypeScript design token exports for programmatic access
- Semantic color system (primary, secondary, accent, success, error, warning, info)
- Color state variants (hover, active, pressed, disabled, subtle)
- Feature colors (magenta, purple, teal)
- Neutral color scale
- Typography tokens (font families, sizes, weights, line heights)
- Spacing scale
- Border radius tokens
- Shadow tokens for elevation
- Animation and transition tokens
- Z-index scale
- Focus ring utilities
- Accessibility support:
  - High contrast mode
  - Reduced motion preference
  - Screen reader utilities
- Theme utilities (setTheme, getActiveTheme, onThemeChange)
- CSS variable name helpers (cssVars object)
