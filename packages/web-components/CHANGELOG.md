# Changelog

All notable changes to @oxi-ui/web-components will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-21

### Added

- Initial release
- `<oxi-button>` - Button web component with variants and sizes
- `<oxi-text-field>` - Text field web component with validation
- React wrapper components for both button and text field
- Angular wrapper components and modules
- Tree-shakable imports for each component
- Full TypeScript support with declaration files
- CSS custom parts for styling
